# Third-Party Notices

PicoPals 8-Bit does not include third-party artwork, character designs, fonts, music, or sound files.

All visible pet sprites are drawn at runtime from original TypeScript/Canvas primitives. Sound effects are synthesized locally with the Web Audio API. Interface symbols use Unicode characters supplied by the player’s operating system and are not bundled assets.

Open-source runtime and development packages are listed in `package.json`; their respective license texts and source links are distributed by npm in each installed package. The application build contains React, React DOM, Dexie, and their required runtime modules under their published licenses.
