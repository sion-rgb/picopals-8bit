# Third-Party Notices

PicoPals 8-Bit does not include third-party artwork, character designs, fonts, music, or sound files.

All visible pet and NPC sprites are drawn at runtime from original TypeScript/Canvas primitives. Sound effects are synthesized locally with the Web Audio API. Main navigation and status symbols use the project's original local SVG pixel icon sprite sheet; no third-party icon artwork is bundled.

Open-source runtime and development packages are listed in `package.json`; their respective license texts and source links are distributed by npm in each installed package. The application build contains React, React DOM, Dexie, Firebase Web SDK, and their required runtime modules under their published licenses. Firebase Emulator tooling and `@firebase/rules-unit-testing` are development-only dependencies.
