# Changelog

## 3.0.0 - 2026-07-22

- 新增離線優先 Firebase Authentication／Cloud Firestore 同步架構及 Emulator 設定。
- 新增 Schema v3、同步佇列、revision、checksum、退避重試、多分頁鎖、雲端快照、同步紀錄與衝突處理。
- 新增完整「離線與同步」設定、主畫面同步狀態、離線資源下載／更新／清除與 Persistent Storage 檢查。
- 重製 20 種原創主寵及六位朋友四階段的像素輪廓、配件與動作。
- 以原創 SVG 像素圖示表取代主要操作欄的 Unicode 圖示。
- 新增下一次成長提示、每日三項星光任務、連續獎勵、十步首次引導及星光朋友圈。
- 更新桌面 45/55 版面、手機單欄版面與固定 4×2 操作矩陣。
- 新增 24 項同步／長期玩法單元驗收、8 項 Firestore Rules 測試及 18 條 Playwright 流程。
